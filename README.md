# Vaishnav Ambilpur - Developer Portfolio

A minimal, high-performance, single-page developer portfolio showcasing my projects, technical stack, experience, and contact information.

## 🚀 Live Demo
Visit the live website here: [vaishnavambilpur.github.io](https://vaishnavambilpur.github.io/)

## ✨ Key Features
- **Minimalist Aesthetic**: Clean single-page layout designed for maximum impact and fast scanning.
- **Dynamic Projects Showcase**: Features pinned fullstack projects (Flow-Q, Semantic Reddit Search, Chatty, and Smart Queue).
- **Infinite Logo Marquee**: Smooth, custom-engineered dual-track infinite marquee for highlighting tech stack logos.
- **Responsive Layout**: Pixel-perfect layout across mobile, tablet, and desktop viewports.
- **GitHub Pages CI/CD**: Automatic deployment via GitHub Actions on every push to the `main` branch.

## 🛠 Tech Stack
- **Framework**: Next.js 15
- **Styling**: Tailwind CSS & DaisyUI
- **CI/CD**: GitHub Actions

## 💻 Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/VaishnavAmbilpur/NextJS-Portfolio-Template.git
   cd NextJS-Portfolio-Template
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

4. **Build for Production / Static Export**:
   ```bash
   npm run build
   ```
   This generates the statically exported site inside the `out/` directory.

## 🚢 Deployment (GitHub Pages)
This repository is configured to build and deploy to GitHub Pages automatically via GitHub Actions:
1. Go to repository **Settings > Pages**.
2. Set **Build and deployment > Source** to **GitHub Actions**.
3. Push updates to the `main` branch to trigger the deploy workflow.
